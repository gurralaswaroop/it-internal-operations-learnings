# RAM Upgrade in Laptops

## Introduction

RAM (Random Access Memory) stores active programs and system processes. Increasing RAM improves multitasking and system responsiveness.

---

## Common Laptop RAM Types

<img width="804" height="397" alt="image" src="https://sc04.alicdn.com/kf/Hc18db3d406884edc86f989efd945f390K.jpg" />

- DDR3 SO-DIMM
- DDR4 SO-DIMM
- DDR5 SO-DIMM

---

## Signs You Need More RAM

- slow system performance
- high memory usage
- system freezing during multitasking
- heavy disk usage

---

## Checking RAM Usage

Operating systems provide memory usage statistics through system monitoring tools.

---

## RAM Upgrade Procedure

1. Power off laptop
2. Remove battery if removable
3. Open bottom cover
4. Locate RAM slot
5. Insert compatible RAM module
6. Close laptop and boot system

---

## RAM Compatibility

Ensure:

- correct RAM generation
- matching voltage
- supported capacity by motherboard

---

## Dual Channel Memory

Using two identical RAM modules enables **dual-channel mode**, improving memory performance.

Example:

- 8GB + 8GB → Dual Channel
- 8GB + 16GB → Flex Mode

---

## Conclusion

Upgrading RAM is one of the most effective ways to improve laptop performance.
