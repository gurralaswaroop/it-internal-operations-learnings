# Laptop Disk Replacement (HDD / SSD)

## Introduction

Storage devices in laptops may fail due to wear, physical damage, or aging. Replacing the disk is one of the most common hardware maintenance tasks.

Modern laptops mostly use **SSD instead of HDD** due to better performance and reliability.

---

## Types of Laptop Storage

### HDD (Hard Disk Drive)

- mechanical storage
- slower read/write speeds
- cheaper per GB

---

### SATA SSD

- faster than HDD
- uses SATA interface
- common in older laptops

---

### NVMe SSD (M.2)

- very high speed
- uses PCIe interface
- used in modern laptops

---

## Signs of Disk Failure

- slow system performance
- frequent OS crashes
- clicking noise from HDD
- file corruption
- operating system not booting

---

## Disk Replacement Process

1. Power off the laptop
2. Disconnect power adapter
3. Remove bottom cover
4. Locate storage drive
5. Remove mounting screws
6. Replace disk with new SSD/HDD
7. Reinstall operating system

---

## Post Replacement Tasks

- install operating system
- restore user data
- update drivers
- check disk health

---

## Recommended SSD Upgrade

Typical upgrades:

| Old Drive | Recommended Upgrade |
|-----------|--------------------|
| HDD 500GB | SSD 500GB |
| HDD 1TB | SSD 1TB |
| SATA SSD | NVMe SSD |

---

## Conclusion

Replacing HDD with SSD significantly improves system performance, boot speed, and reliability.
