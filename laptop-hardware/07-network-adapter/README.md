# Laptop Network Adapter

## Introduction

The network adapter allows laptops to connect to networks using:

- Ethernet
- Wi-Fi

---

## Types of Network Adapters

### Ethernet Adapter

Provides wired network connectivity.

Advantages:

- stable connection
- low latency
- high reliability

---

### Wi-Fi Adapter

Provides wireless connectivity.

Common standards:

- Wi-Fi 5 (802.11ac)
- Wi-Fi 6 (802.11ax)

---

## Network Adapter Issues

Common problems:

- no internet connection
- driver missing
- weak signal
- network disconnections

---

## Troubleshooting

Steps:

1. check network drivers
2. reset network settings
3. update network adapter driver
4. check router connectivity

---

## Conclusion

Network adapters enable system connectivity and are essential for internet access and communication.
